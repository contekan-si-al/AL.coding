Cara menjalankan: *Windows + VS Code (Anaconda) + Terminal Git Bash
1. Set up virtual environment
    python -m venv venv
    source D:/AL/venv/Scripts/activate (D:/AL/ bisa disesuaikan dengan lokasi penyimpanan folder)

2. Install Dependency
    pip install -r requirements.txt

3. Setup environment
    - Buat file .env dengan isi:
      SPREADSHEET_ID=spreadsheet_id_kamu
      * Spreadsheet ID adalah teks antara /d/ dan /edit pada URL. Contoh: https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit
    - Cara lain bisa dengan membuka berkas ".env.example" | Replace "spreadsheet_id_kamu" dengan hasil copy "SPREADSHEET_ID" | jalankan kode "cp .env.example .env" di terminal
    - Siapkan google-sheets-api.json untuk Google Sheets
      * Cara mendapatkan berkas sudah dipelajari di dalam Modul. Hanya mengganti nama hasil unduhan ke "google-sheets-api"
4. Run ETL script
    python main.py

5. Run unit test
    pytest tests/ -v

6. Run test coverage
    pytest tests/ --cov=tests --cov-report=term-missing
